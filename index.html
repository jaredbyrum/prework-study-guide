HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element of the website is required for SEO, Search Engine Optimization.
  • The Style element defines the CSS styles.
  • The meta element defines metadata such as character sets, descriptions, keywords, author, and viewport.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • DRY, Dont Repeat Yourself! save complexity and repetitiveness.

GIT

  • git status: checks what branch we are currently in
  • git checkout branch name: switches to a desired branch
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: add, adds modifications to the staging area. -A flad indicates we want to add all changes.
  • git commit -m "desired note": commits changes. -m adds commit message of whats being commited.
  • git pull origin main: pull, recieves a branches mods into enviro. origin, the source were pulling from.
  • git push origin branch: push, pushes changes made locally to the remote branch.

JavaScript

  • A varialble is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
